# Sound Effects for Football IQ

Place the following MP3 files in this folder:

| File | Description | Suggested Source |
|------|-------------|------------------|
| `correct.mp3` | Right answer - short positive chime | [Kenney UI Audio](https://kenney.nl/assets/ui-audio) |
| `incorrect.mp3` | Wrong answer - short buzzer/error sound | [Kenney UI Audio](https://kenney.nl/assets/ui-audio) |
| `tap.mp3` | Button press - subtle click | [Kenney Interface Sounds](https://kenney.nl/assets/interface-sounds) |
| `level_up.mp3` | Level up/unlock - celebratory fanfare | [Kenney Interface Sounds](https://kenney.nl/assets/interface-sounds) |
| `tick.mp3` | Timer countdown - single tick | [Kenney UI Audio](https://kenney.nl/assets/ui-audio) |
| `whistle.mp3` | Game over - referee whistle | [Pixabay](https://pixabay.com/sound-effects/search/whistle/) |

## Recommended Downloads

### Kenney.nl (CC0 - No Attribution Required)
1. Download [UI Audio Pack](https://kenney.nl/assets/ui-audio)
2. Download [Interface Sounds](https://kenney.nl/assets/interface-sounds)

### Pixabay (Royalty-Free)
Search for specific sounds like "whistle" or "quiz correct"

## Notes
- All sounds should be short (under 2 seconds)
- Keep file sizes small for mobile performance
- The app will gracefully fail if sounds are missing
